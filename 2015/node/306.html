5. - 7.6.2015, Bratislava

DrupalCampCS
Three days of Drupal action!

Training: Services 101

Services and APIs - what would we do without them? This workshop will cover basics of building services with Drupal, how to communicate between sites, send and consume requests and make Drupal kittens happy.

This will be a hands-on session, so please bring your computers, with Acquia Dev Desktop installed.

Requirements:

  • Computer
  • Acquia Dev Desktop installed
  • Basic PHP programming skills
  • Basic knowledge of Drupal API

Language: Slovak

Level

  • Beginner
Timeframe
Sun, 06/07/2015 - 10:00

Room

  • Training room A
